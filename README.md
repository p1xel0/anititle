# anititle
Get the title of anime files using anitomy.

## Example Usage:
Input: ```anititle '[SubsPlease] Dragon Quest - Dai no Daibouken (2020) - 39 (1080p) [57E3D085].mkv'```

Output: ```Dragon Quest - Dai no Daibouken (2020)```

## Build Instructions:
```
cd /path/to/anititle/
git submodule update --init
cmake .
make
```
